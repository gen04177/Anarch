# Anarch PS5 Port

`hbldr anarch_sdl.elf -h` or `hbldr anarch_sdl.elf`

## Controls:

- Arrows: movement
- [L3]: free look
- [R3]: shoot
- [R1]: next weapon
- [L1]: previous weapon
- [CROSS]: jump
- [CIRCLE]: cancel, strafe
- [SQUARE]: shoot, confirm
- [TRIANGLE]: (+ down = menu, + up = jump, ...)
- Options: map
- Touchpad: menu
